## 中间件的特点：

1. koa对象通过use方法加入一个中间件
2. 一个中间件就是一个函数
3. 中间件的执行顺序符合洋葱模型
4. 内存中间件能否执行取决于外层中间件中的next函数是否调用
5. 调用next函数得到的是Promise对象
app.use(async(ctx, next)=>{
    // 刚进入中间件想做的事情
    await next()
    // 内层所有中间件结束之后想做的事情
})
